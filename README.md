# bittsin_api
[bitts.in](http://bitts.in/ref/blackout314) api wrapper | [payout](http://bitts.in/payout-rates)

```php
$KEY      = 'YOUR-KEY';
$bittsin  = new bittsin($KEY);
$link     = $bittsin->__get('http://www.google.com');
```

response
```
 {"status":"success","shortenedUrl":""http:\/\/bitts.in\/xxxxx""} 
```
